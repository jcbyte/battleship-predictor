A simple web app for showing the best strategy to win battleships.

## Production Build

https://battleship-predictor.vercel.app/

## Run Locally

Install dependencies

```bash
npm install
```

Start postcss

```bash
npm run dev-css
```

Start local development server

```bash
npm run dev
```

## Tech Stack

React, Typescript, TailwindCSS
